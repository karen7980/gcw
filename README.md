Objetivo
- Este site tem como objetivo informar as pessoas sobre a história por trás de Wicked, o musical.<br/>

Telas
- Este site apresenta uma tela de menu e uma página para cada personagem principal da trama, Elphaba e Glinda.<br/>

Framework
- Este site foi construído usando Materialize.<br/>

LP
- HTML, CSS E JavaScript.<br/>

Cronologia
- Este site foi feito num período de 2 semanas.<br/>

Dedicação
